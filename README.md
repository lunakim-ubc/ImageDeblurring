# Image Deblurring

MATH 441 Project3 — 2025/26 W2

Deblurring blurred images using Tikhonov regularization and related spectral filtering methods (Naive inverse, Truncated SVD). Covers linear motion blur, color image blur, and 3D volumetric blur.

- Umay Gokturk

## Requirements

- Python 3.x
- NumPy
- SciPy
- Matplotlib
- Pillow
- CVXPY

## Notebooks

- `motion_deblur_linear.ipynb` — Linear motion blur and deblurring
- `colorblurring.ipynb` — Color image blur and deblurring
- `3D_blur.ipynb` — 3D volumetric blur and deblurring

## Usage

See individual notebooks for each blur type.
